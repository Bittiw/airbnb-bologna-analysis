# Airbnb Market Analysis — Bologna

This project analyzes Airbnb listings in **Bologna** to identify key drivers of price,
occupancy patterns, and seasonal demand using Python and exploratory data analysis.

---

## Project Questions

1. Which features most influence Airbnb prices?
2. What differentiates high-occupancy listings from low-performing ones?
3. How do location and seasonality affect demand?

---

## Key Insights

- Most listings are priced below €200 per night (right-skewed distribution).
- Prices increase as distance to Bologna city center decreases.
- Professional hosts tend to charge higher prices and achieve higher occupancy.
- Demand peaks during spring and summer months.
